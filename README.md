 Programación 2

Aquí estaré subiendo todos mis proyectos y tareas de Programación 2 :D

 Tarea 1 - Jerarquía de Clases

- Descripción: Implementación de herencia con clases como Persona, Estudiante, Profesor, Empleado, etc.
- Carpeta: prog/Tarea1/Tarea1
- Archivos principales:
  - Program.cs (punto de entrada)
  - Tarea1.csproj
  - Entities/ (con las clases)

 Cómo ejecutar el proyecto

1. Clona el repositorio:
git clone https://github.com/Ranleisy/Programacion-2.git

2. Entra a la carpeta del proyecto:
cd Programacion-2/prog/Tarea1/Tarea1

3. Compila y ejecuta:
dotnet build
dotnet run

O simplemente abre el archivo `Tarea1.csproj` en Visual Studio y presiona F5.

Tecnologías usadas:  
C# · .NET · Programación Orientada a Objetos
